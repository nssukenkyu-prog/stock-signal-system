# 免責事項・注意事項

## ⚠️ 重要な注意事項

### 1. 投資助言ではありません

本システム（Stock Signal Bot）の出力は、**情報提供のみを目的**としており、投資助言・推奨・勧誘ではありません。

- 本システムが生成するシグナル（BUY/SELL/HOLD/WATCH）は、あくまで技術的指標に基づく参考情報です
- 最終的な投資判断は、ご自身の責任において行ってください
- 本システムの利用により生じた損失について、開発者は一切の責任を負いません

### 2. 過去のパフォーマンスは将来を保証しません

- バックテスト結果は過去のデータに基づくものであり、将来の収益を保証するものではありません
- 市場環境の変化により、システムのパフォーマンスは大きく変動する可能性があります
- 実際の取引では、スリッページや手数料により結果が異なる場合があります

### 3. リスクについて

株式・ETF・投資信託への投資には以下のリスクが伴います：

- **価格変動リスク**: 市場価格は常に変動し、投資元本を下回る可能性があります
- **為替リスク**: 外国証券への投資は、為替変動の影響を受けます
- **流動性リスク**: 市場状況により、希望する価格で売買できない場合があります
- **システムリスク**: 技術的問題により、通知の遅延・欠落が発生する可能性があります

### 4. システムの限界

- 本システムはAI/機械学習を使用せず、ルールベースの統計モデルに基づいています
- ニュース・イベントへの反応は限定的です（キーワードスコアリングのみ）
- 急激な市場変動や予期せぬイベントには対応できません
- データソース（Yahoo Finance等）の障害時はシグナル生成が停止します

## 📋 データの取り扱い

### 収集するデータ
- 価格データ（公開情報）
- 通知送信ログ（いつ、何を送信したか）
- 保有銘柄情報（ご自身で入力したもの）

### データの保管
- すべてのデータはCloudflare上に保存されます
- 第三者への提供は行いません
- ご自身でいつでもデータを削除できます

### ログの保持期間
- 通知ログ: 90日間
- シグナル履歴: 無期限（手動削除可能）
- 価格データ: 直近1年分

## 🔒 セキュリティ

### APIキーの管理
- LINE Channel Access TokenはCloudflare Secretsで暗号化保存されます
- トークンは第三者に共有しないでください
- 不正利用の疑いがある場合は、直ちにトークンを再発行してください

### アクセス制限
- 本システムのAPIエンドポイントは認証なしでアクセス可能です
- 機密性の高い操作（データ削除等）は実装していません
- 必要に応じてCloudflare Access等で保護してください
